Генератор рамки на ассемблере
===

Кратко
---

Данная программа принимает на вход в качестве аргумента коммандной строки стиль рамки, записанный 16-ричными числами и сообщение, котрое выводится на экране, обрамленное рамкой, отцентрированной по горизонтали и вертикали.

Программа разрабатывалась под операционную систему MS DOS, на ассемблере процессора x86.
